Description

The Attributes of a BASE are stable, persistent data and information about it. This module offers all the functions that a developer needs to read and write to attributes. Attributes are like a book with pages, you write to a page with a key and a value.

write/4

write(Page, Key, Value, BH) -> ok

  • Page = binary()
  • Key = binary()
  • Value = term()
  • BH = base_handle()

Page: The binary representing the specific attributes page to write to.
Key: The binary key for the attribute to write.
Value: The value to write for the attribute.
BH: The base handle that identifies the backend where the attribute will be written.

returns: Returns ok if the write operation is successful, otherwise an error tuple.

Writes an attribute to a specific attributes page.

This function writes an attribute with the given Key and Value to the Page specified. The BH parameter is a base handle that identifies the backend where the attribute will be written.
