subscribe/4

subscribe(Page::binary(), Key::binary(), Subscriber::pid() | module() | agent_handle(), BH::base_handle()) -> CurrentValue::term()

Subscribe to a variable on a Page with either a module, PID, or AgentHandle. The module needs to export var_update(Page,Key,Value,BH) the process with PID will receive a message of #base_var_update{} The agent will call the handle_message() callback of the plugin for it.
